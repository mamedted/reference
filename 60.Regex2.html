Regular Expression

Creating a regex

/pattern/

let regex2 = new RegExp('pattern')

Methods

test() returns true/false

match() returns array of matches or null

replace() replace matched patterns

search() returns index of first match

Flags

/pattern/flags

g find all matches

i case insensitive

m multiline node

s dot all

Character classes

SPECIAL CHARACTER

. maches any character except new line

\d matches all digit

\D matches all non digit

\w matches letter/numberic character

\W matches non letter/numberic character

\s matches whitespace(space,tab,newline)

\S matches non-whitespace

CUSTOM CHARACTER

[abc] matches a, b, or c

[a-z] matches any letter from a to z

[^abc] matches anything excpet a, b, and c

Quantifiers

* 0 or more times

+ 1 or more times

? 0 or 1 times, optional

{n} exactly n times

{n,} n or more times

{n,m} between n and m times
